# 🧬 BIOCARE-BIOTECH

Application de gestion des alertes pour laboratoire de microbiologie.

## 🚀 Accès rapide

**[➡️ Ouvrir l'application](https://VOTRE_USERNAME.github.io/biocare-biotech)**

## 📱 Installation mobile

1. Ouvrir le lien sur Chrome/Safari mobile
2. Menu → "Ajouter à l'écran d'accueil"
3. L'application s'installe comme une app native !

## 🔧 Fonctionnalités

- ✅ Signalement de produits manquants/périmés
- ✅ Synchronisation temps réel Firebase
- ✅ Mode hors-ligne fonctionnel
- ✅ Export Excel des signalements
- ✅ Gestion des utilisateurs (Propriétaire)
